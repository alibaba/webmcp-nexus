- **入口与导出**：`src/index.ts` 统一导出核心 API `registerGlobalTools`（全局一次性注册）和 `useWebMcpTools`（React Hook 生命周期绑定），以及底层类型定义。
- **环境适配层**：`polyfill.ts` 负责惰性初始化 `@mcp-b/webmcp-polyfill`，确保在非原生支持 MCP 的浏览器中存在 `navigator.modelContext`。
- **核心注册中心**：`registry.ts` 是模块的核心，实现了多作用域（global/component）的工具所有权管理、`AbortController` 生命周期控制，以及针对 Chrome 原生实现与 Polyfill 差异的 `patchModelContextEventSupport` 兼容逻辑。
- **构建配置**：使用 `tsup` 进行打包，生成 ESM/CJS 双格式及类型声明，并将 `react` 设为外部依赖以适配不同宿主环境。
