- 采用 React Router 实现页面路由（`/`, `/todos`, `/editor`），并在 `App.tsx` 中通过嵌套 Context Provider（Canvas, Todo, Editor）管理各业务域状态。
- 状态管理层位于 `src/store`，使用 React Context API 封装独立的数据逻辑（如 `TodoStore.tsx` 提供 CRUD 操作），并通过自定义 Hooks（如 `useTodoStore`）向 UI 层注入能力。
- 工具桥接层位于 `src/tools`，通过 `navigation-bridge.ts` 将路由跳转能力发布为全局可访问的引用，供 `webmcp-nexus-sdk` 注册的工具（`navigation.ts`）调用，实现 AI Agent 对前端导航的控制。
- 构建系统支持双引擎：默认使用 Vite（`vite.config.ts`）进行开发与打包，同时保留 Webpack 配置（`webpack.config.ts`）以兼容特定插件生态。
