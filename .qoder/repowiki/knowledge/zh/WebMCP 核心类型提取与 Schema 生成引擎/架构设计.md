- **入口层**：`src/index.ts` 统一导出 `transformCode`（构建时变换入口）及底层 API (`extractToolsFromFile`, `generateSchema`)。
- **变换协调层**：`src/transform.ts` 负责检测源码中的 `registerGlobalTools` 或 `useWebMcpTools` 调用，协调提取与注入流程。
- **静态分析层**：`src/ts-extractor.ts` 利用 `ts-morph` 逆向追踪函数定义、解析 JSDoc 注释及参数类型，支持对象字面量解构和 Namespace Import 两种注册模式。
- **模式映射层**：`src/schema-generator.ts` 将提取的 TypeScript 类型信息（如联合类型、可选属性）映射为标准的 JSON Schema 结构，并生成运行时代码注入字符串。
- **依赖方向**：变换逻辑依赖提取器，提取器依赖模式映射器，形成单向数据流。
